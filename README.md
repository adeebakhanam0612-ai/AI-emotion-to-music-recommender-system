🎵 AI Emotion Music Recommender System

📌 Project Overview

The AI Emotion Music Recommender System is an Artificial Intelligence-based application that detects a user's facial emotion from an uploaded image or live camera capture and recommends suitable songs according to the detected mood.

The project combines Computer Vision, Machine Learning, and Music Recommendation concepts to provide a personalized and interactive user experience.

---

✨ Features

✅ Emotion Detection from Uploaded Images

✅ Real-Time Emotion Detection using Live Camera

✅ AI-Based Facial Emotion Recognition

✅ Mood Confidence Score Visualization

✅ Emotion-wise Music Recommendation

✅ Interactive and Attractive Streamlit Interface

✅ Fast and User-Friendly Experience

🚀 Working Principle

1. User uploads an image or captures a live photo.
2. The FER model analyzes facial expressions.
3. The dominant emotion is detected.
4. Confidence scores for emotions are generated.
5. Music recommendations are provided based on the detected emotion.
6. The user can explore songs matching their mood.
